# poker-chips-bot
Poker chips calculator Telegram bot

<h1>1. Start</h1>
<h3>Firstly to find the bot you can search by bot username @PokerChipsBot
Then, after opening the bot, you will see this screen</h3>

<img src="https://user-images.githubusercontent.com/62104475/97810417-e0afe880-1c9d-11eb-979a-e0de50acbbab.jpg" width="300" height="500">

<h1>2. First steps</h1>
<h3>Then you have 2 options, start with the default value of chips (1000) or set your amount of chips by clicking on the button "указать количество фишек"</h3>

<img src="https://user-images.githubusercontent.com/62104475/97810608-49e42b80-1c9f-11eb-9a6c-bceb051fdaa1.jpg" width="300" height="500">

<h1>3. Setting chips value</h1>
<h3>After clicking "указать количество фишек", to set your chips value, you should write the number you have initially, and press enter, after will appear a markup button</h3>

<img src="https://user-images.githubusercontent.com/62104475/97810634-8fa0f400-1c9f-11eb-8f6c-10970298f564.jpg" width="300" height="500">

<h1>4. Your turn</h1>
<h3>Like in real poker, you have 3 options: Bet, Check and Fold. Press one of them, to continue the game</h3>

<img src="https://user-images.githubusercontent.com/62104475/97810682-f32b2180-1c9f-11eb-93d1-444ea877a639.jpg" width="300" height="500">

<h1>5. Results?</h1>
<h3>Okay, let's imagine you have good pair of cards, and you want to bet. After clicking the bet button, you should write your bet number, how much you want to bet. This sum will be deducted from your stack. After entering, you will appear a menu below. You can continue the game, or choose that you won the game or lose</h3>

<img src="https://user-images.githubusercontent.com/62104475/97810840-ef4bcf00-1ca0-11eb-960d-313881bebef2.jpg" width="300" height="500">

<h1>6. Repeat round</h1>
<h3>Maybe you win, maybe not, but after results will appear the "start new game" button, so you can play round one more time!</h3>

<img src="https://user-images.githubusercontent.com/62104475/97810987-e3144180-1ca1-11eb-857a-1962e6326011.jpg" width="300" height="500">

<h1>Enjoy!</h1>
